# Dashboard de Análise de Carros Usados

## Descrição
Este aplicativo web permite explorar dados de anúncios de carros usados nos EUA através de visualizações interativas.

## Funcionalidades
- Histogramas interativos
- Gráficos de dispersão
- Filtros para explorar os dados

## Como executar
1. Clone o repositório
2. Instale as dependências: `pip install -r requirements.txt`
3. Execute: `streamlit run app.py`

*Link do aplicativo será adicionado após o deploy*
